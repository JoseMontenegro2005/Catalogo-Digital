# Catálogo Interactivo - Distrimayor Pasto

Este proyecto es una aplicación web de alto rendimiento construida con **React**, que digitaliza el catálogo de laboratorios de Distrimayor Pasto separado en 3 categorias (Drogueria, Tradicional, Belleza). El objetivo es ofrecer una herramienta de ventas moderna, rápida y con una experiencia de usuario fluida.

**[Ver la demo en vivo](https://catalogo-distrimayor.vercel.app/)**

---

## Características Principales

* **Filtro en Tiempo Real:** Permite a los usuarios buscar productos y ver resultados al instante. Esta funcionalidad está construida con el manejo de estado de React (`useState`) para una interfaz reactiva y sin recargas.
* **Componentes Reutilizables:** La interfaz está construida usando componentes de React (ej. `<ProductCard>`, `<SearchBar>`), lo que hace el código limpio, mantenible y escalable.
* **Renderizado de Next.js:** Aprovecha Next.js para un rendimiento de carga inicial superior y una mejor optimización (SEO).
* **Diseño 100% Responsive:** La interfaz se adapta perfectamente a cualquier tamaño de pantalla, desde teléfonos móviles hasta monitores de escritorio.

---

## Tecnologías Utilizadas

* **[React](https://reactjs.org/)**: Biblioteca principal para construir la interfaz de usuario basada en componentes.
* **[Next.js](https://nextjs.org/)**: Framework de React para producción (usado para el renderizado, enrutamiento y optimizaciones).
* **React Hooks**: Uso de `useState` para el manejo del estado local (como el texto de búsqueda).
* **CSS3**: Para el diseño y layout.
* **Vercel**: Plataforma de despliegue optimizada.

## Cómo ejecutar este proyecto localmente

1.  Clona este repositorio:
    ```bash
    git clone https://github.com/JoseMontenegro2005/Catalogo-Digital.git 
    ```
2.  Navega a la carpeta del proyecto:
    ```bash
    cd catalogo-distrimayor
    ```
3.  Instala las dependencias:
    ```bash
    npm install
    # o
    # yarn install
    ```
4.  Ejecuta el servidor de desarrollo:
    ```bash
    npm run dev
    # o
    # yarn dev
    ```
5.  Abre [http://localhost:3000](http://localhost:3000) en tu navegador para ver la aplicación.
